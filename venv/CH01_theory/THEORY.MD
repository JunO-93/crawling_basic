    ✔ 역사 : 검색을 위해 크롤링이 만들어 졌다. 홈페이지글들의 글을 구글이 모두 자기 서버로 가져와 검색엔진으로 조회할 수 있게 만들었다.
    ✔ 활용처 : 
        (1). 사이트에서 원하는 데이터 추출하기
        (2). 업무 자동화하기
        (3). 알림 받기
        
    - 인터넷과 웹( World wide web )                 
    - 웹서버와 디비
    ✔
    - http와 소켓
    - 브라우저
    - 웹앱과 api
