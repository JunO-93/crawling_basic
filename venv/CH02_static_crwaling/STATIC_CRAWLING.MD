# 정적크롤링
```buildoutcfg
정적 크롤링 vs 동적 크롤링

- 정적 크롤링
    * 크롤링 속도 : 빠르고 간단함
    * 개발 편의성 : 처음엔 쉽고 고도화 될수록 힘듦
    * 디버깅 편의성 : 테스트 쉬움
    * 오류 취약점 : 상대적으로 낮음
    
- 동적 크롤링
    * 크롤링 속도 : 느리고 복잡함
    * 개발 편의성 : 처음엔 손이 많이가지만 나중에 편리함
    * 디버깅 편의성 : 테스트 어려움
    * 오류취약점 : 상대적으로 높음
```
